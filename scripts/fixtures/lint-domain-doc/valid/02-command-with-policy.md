# Valid fixture 02: コマンド + ポリシー（駆動契機なし）

## プラン集約

### コマンド

```
プランニングを開始する:
    入力: チケットID AND 開発者
    イベント: プランニングが開始された / プランニング中プラン
    失敗時:
        - チケット未トリアージ

判断依頼の発行を反映する:
    入力: プランニング中プラン
    イベント: 判断依頼中プラン

実装プランを再開する:
    入力: 判断依頼中プラン AND 回答内容
    イベント: プランニング中プラン

日次集計を実行する:
    入力: 集計対象日
    イベント: 日次集計が完了した
```

### ポリシー

```
プラン再開ポリシー:
    きっかけ: 回答が受理された
    実行するコマンド: 実装プランを再開する
```
